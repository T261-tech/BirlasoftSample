Git Commands:

git init- To initialize the git local repo
git status- to check the status of the files tracked/untracked
git add- add all the files to be tracked
git remote and origin<url>- to connect the remote 
	 repo(github repo) and the local repo(git repo)
git commit -m "message"- to save files in the local copy(git)
git push origin master- to push the commited changes to the remote copy.
